# Weather App

## Description

This is a beginner-level Python Weather Application that fetches current weather information using the WeatherAPI.

Users can enter a city name or ZIP code to view:

- Temperature
- Humidity
- Weather Condition
- Wind Speed
- Region
- Country

## Features

- Current weather information
- Search using city or ZIP code
- Error handling for invalid locations
- Internet connection handling
- Simple command-line interface

## Technologies Used

- Python
- Requests Library
- WeatherAPI

## Installation

Install the required package:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python weather.py
```

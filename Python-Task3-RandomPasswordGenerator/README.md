# Random Password Generator

## Project Description

This is a beginner-friendly command-line Python application that generates random passwords based on user preferences. The user can choose the password length and whether to include letters, numbers, and symbols.

## Features

- Accepts password length from the user
- Allows users to include:
  - Letters
  - Numbers
  - Symbols
- Generates a random password
- Validates password length
- Validates that at least one character type is selected
- Handles invalid numeric input using `try-except`

## Technologies Used

- Python 3
- random module
- string module

## How to Run

1. Open the project in Visual Studio Code.
2. Run the Python file.
3. Enter:
   - Password length
   - Whether to include letters
   - Whether to include numbers
   - Whether to include symbols
4. The program generates a random password.

## Example Output

```

Include letters? (y/n): y
Include numbers? (y/n): y
Include symbols? (y/n): y

Generated Password = A8@mK2#P
```

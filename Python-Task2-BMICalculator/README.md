# BMI Calculator

## Description

A simple Python command-line application that calculates Body Mass Index (BMI) based on a user's weight and height. The program also classifies the BMI into health categories.

## Features

* Accepts weight (kg) and height (m) as input
* Validates user input
* Handles non-numeric input gracefully
* Calculates BMI
* Displays BMI up to two decimal places
* Classifies BMI as:

  * Underweight
  * Normal Weight
  * Overweight
  * Obese

## Technologies Used

* Python 3
* Visual Studio Code

## BMI Formula

BMI = Weight (kg) / Height² (m²)

## How to Run

1. Open the project in Visual Studio Code.
2. Run the Python file.
3. Enter your weight and height.
4. View your BMI and category.
